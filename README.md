#  Trufa Store

Sistema de gerenciamento de vendas de trufas desenvolvido com Spring Boot, MySQL e integração Frontend + Backend.

---

#  Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Criação de vendas
- Integração com banco de dados MySQL
- API REST com Spring Boot
- Frontend integrado ao backend

---

#  Tecnologias Utilizadas

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- HTML
- CSS
- JavaScript
- Maven

---

#  Banco de Dados

O sistema utiliza MySQL para armazenamento dos dados de:
- Produtos
- Vendas
- Itens da venda

---

#  Como executar o projeto

## Backend
1. Abrir o projeto no Eclipse
2. Configurar o MySQL
3. Executar a classe:
```

TrufaStoreApplication.java

```

---

## Frontend
1. Abrir o arquivo:
```

index.html

```

2. Utilizar normalmente no navegador

---

- realizar vendas
